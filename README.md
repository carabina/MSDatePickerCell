## Requirement
- Xcode 8
- Swift 3
- iOS 8.0 or later

## Usage
```Swift
let cell = MSDatePickerCell(style: .YMD) { (date: Date) in
  // Do Something
}
return cell
```

## Installation
#### [CocoaPods](https://cocoapods.org/)
Add the following line to your Podfile:
```ruby
use_frameworks!

target 'YOUR_TARGET_NAME' do
  pod "MSDatePickerCell"
end
```
